# iOS — File Links & @References

> iOS counterpart of NB-20

**Scope:** Cross-referencing between chat messages and workspace

## Message Links

- `[[file:...]]` and `[[canvas]]` links rendered as tappable `AttributedString` spans
- File links styled with green tint, canvas links with blue tint
- Tapping navigates via `TabView` selection binding

## @Autocomplete

- Custom dropdown overlay on input field triggered by `@` character
- Lists workspace file paths
- Selection inserts the reference into the message

## Dependencies

- NB-28 (iOS Navigation) — tab switching
- NB-31 (iOS Files Tab) — file deep linking
