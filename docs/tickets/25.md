# Connection Management Sheet

**Scope:** Bottom sheet opened from agent bar server icon

## Introduces Reusable `Sheet` Component

- Slide-up bottom sheet with backdrop
- Used by this ticket, NB-26 (Bot Profile), and NB-27 (Settings)

## WebSocket Server

- URL input field for WebSocket server address

## Actions

- **Save & Reconnect** (primary button): saves URL and reconnects
- **Test Connection** (secondary button): pings the server and displays:
  - Latency (ms)
  - Model name
  - Uptime

## System Info (Read-Only)

- Host
- Model
- Backend
- Gateway URL

## Capabilities

Tag pills showing available bot capabilities:

- File I/O, Web search, Shell exec, Subagents, Cron, Memory, Multi-channel

## Dependencies

- NB-18 (Navigation Overhaul) — agent bar connection icon trigger
