# Enhanced Logs & Split Pane

**Scope:** Replaces current `DebugPanel` with split-pane layout

## Split Pane Layout

- **Top pane**: System Monitor (NB-23)
- **Bottom pane**: Logs
- **Pane cycling**: click header to cycle through collapsed → half → full states
  - Monitor header click expands monitor pane
  - Logs header click expands logs pane

## Enhanced Log Viewer

- **Filter input**: search across type, body, and direction fields
- **Copy**: button to copy all visible (filtered) logs to clipboard
- **Log count badge**: shows number of visible entries
- **Clear button**: clears the log buffer

## Color Coding

| Type | Color |
|------|-------|
| message | Blue |
| response | Green |
| event | Yellow |
| stream | Purple |
| error | Red |

## Frame Display Format

```
HH:MM:SS → [type] body
```

- Body truncated to 120 characters
- Monospace font

## Dependencies

- NB-23 (System Monitor) — embedded in top pane
- NB-18 (Navigation Overhaul) — Logs tab
