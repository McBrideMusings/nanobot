# Enhanced Bot Profile & Dynamic Status

> **Supersedes NB-7** — covers all original Bot Profile scope plus dynamic bot-updatable status fields, 3D avatar integration, and bottom sheet presentation

**Scope:** Bottom sheet opened from agent bar avatar

## Profile Header

- Avatar (3D via NB-19, or static icon fallback) with status ring
- Bot name ("PierceTower")
- Version string

## Live Status Card

Bot-updatable fields displayed as key-value rows:

- **Status** — e.g. "Online", "Busy", "Away"
- **Current task** — e.g. "Reviewing pull requests"
- **Mood** — e.g. "Focused", "Curious"
- **Last active** — relative timestamp

## Protocol Extension

Bot pushes status updates via:

```json
{
  "type": "event",
  "category": "meta",
  "event": "status_update",
  "data": {
    "status": "busy",
    "current_task": "Reviewing PRs",
    "mood": "focused"
  }
}
```

- Partial updates merge with existing state
- Automatic updates from existing events (e.g. `thinking_started` → "Thinking...")

## Dependencies

- NB-19 (3D Avatar) — avatar display
- NB-25 (Connection Sheet) — introduces Sheet component
- NB-18 (Navigation Overhaul) — agent bar avatar trigger
