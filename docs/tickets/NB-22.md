# iOS — 3D Agent Avatar

> iOS counterpart of NB-19

**Scope:** SceneKit or RealityKit animated robot head

## Geometry

- Metallic sphere head
- Glowing eyes (blue default, yellow when thinking)
- Antenna with pulsing green tip
- Ear accents

## Animations

- `SCNAction` sequences for idle sway, blink, pulse
- Status-reactive eye and antenna colors driven by `agentStatus`

## Integration

- Used in agent header bar (small) and profile sheet (large)
- Settings toggle to disable (falls back to SF Symbol icon)

## Dependencies

- NB-28 (iOS Navigation) — header bar placement
