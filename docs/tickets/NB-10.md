# UI Navigation Overhaul

> **Supersedes** current `SettingsBar` + sidebar toggle layout

**Scope:** Foundation ticket — must land before all other prototype UI tickets

## Desktop Layout (≥768px)

- **Activity bar**: 52px fixed left column — vertical icon strip
- Tabs: Chat, Canvas, Files, Logs
- Active tab highlighted with accent color + left border indicator

## Mobile Layout (<768px)

- Activity bar hidden
- **Bottom tab bar**: same four tabs (Chat, Canvas, Files, Logs)
- Native-feeling tab bar with icons + labels

## Agent Bar

Always-visible 40px top header across all tabs:

- **Bot avatar** (with status ring overlapping below bar) — tappable, opens Bot Profile sheet (NB-26)
- **Bot name** + status text
- **Connection icon + status dot** — tappable, opens Connection sheet (NB-25)
- **Settings gear** — tappable, opens Settings sheet (NB-27)

## New Components

- `ActivityBar.tsx` — desktop left rail
- `TabBar.tsx` — mobile bottom tabs
- `AgentBar.tsx` — universal top header

## Removes

- `SettingsBar.tsx` — functionality migrates to agent bar + sheets
