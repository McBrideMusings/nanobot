# System Monitor — GPU & CPU

**Scope:** nvtop/btop-inspired monitoring panel

## GPU Tab

- **Device headers**: GPU name, type, PCIe info, clock speeds, temperature, power draw
- **Utilization bars**: `│` fill character style, percentage labels
- **Time-series chart**: SVG with gradient fill showing GPU utilization over time
- **Process table**: PID, USER, DEV, TYPE, GPU%, MEM, CMD columns

## CPU Tab

- **Per-core bar grid**: 2-column layout, 24-color palette for core identification
- **CPU/RAM utilization bars**: system-wide usage
- **Time-series chart**: SVG with per-core traces
- **Process table**: similar to GPU but with CPU-specific columns

## UI

- GPU/CPU toggle tabs at top
- Monospace font throughout
- Responsive layout

## Phased Implementation

- **Phase 1**: Fake data via `useFakeTimeSeries` hook for UI development
- **Phase 2**: Real data via WebSocket from backend

## Reusable Components

- `UtilBar` — horizontal utilization bar with label
- `NvtopChart` — SVG time-series chart with gradient fill
- `useFakeTimeSeries` — hook generating realistic fake monitoring data

## Dependencies

- NB-18 (Navigation Overhaul) — tab infrastructure
