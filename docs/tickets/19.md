# 3D Agent Avatar

**Scope:** Three.js animated robot head component

## Geometry

- Metallic dark sphere head
- Glowing blue eyes (yellow when thinking)
- Antenna with pulsing green tip
- Ear accents

## Animations

- Idle sway (subtle lateral rotation)
- Subtle vertical bob
- Realistic blinking
- Antenna glow pulse

## Props

- `size` — pixel dimensions (46px in agent bar, 64px in profile sheet)
- `agentStatus` — drives eye/antenna color changes

## Fallback

- Settings toggle to disable 3D avatar
- Falls back to static SVG icon when disabled

## Dependencies

- `three` npm package
- NB-18 (Navigation Overhaul) — agent bar placement
