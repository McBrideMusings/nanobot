# iOS — Enhanced Files Tab

> iOS counterpart of NB-21

**Scope:** SwiftUI file browser with full CRUD operations

## Navigation

- SwiftUI `List` with `NavigationStack` for hierarchical file browsing
- Breadcrumb-style navigation via navigation title and back buttons

## Search & Filter

- `.searchable` modifier for file name and content search
- Sort and filter controls

## Context Menus

- `.contextMenu` modifier for rename, duplicate, delete, new file, new folder actions

## File Editor

- `TextEditor` view for editing file contents
- Save support with dirty tracking

## Deep Linking

- Chat file links (NB-30) navigate directly to the file in this tab

## Dependencies

- NB-28 (iOS Navigation) — tab infrastructure
