# Enhanced Files Tab

> **Supersedes NB-6** — covers all original Workspace Inspector scope plus search, sort, filter, context menus, inline rename, and deep linking

**Scope:** Major expansion of file browsing and editing

## Breadcrumb Navigation

- Path display: `~ / memory / ...` with clickable segments
- Each segment navigates to that directory level

## Unified Toolbar

- Breadcrumb path
- **Search**: toggle between name search and content search modes
- **Sort**: name / size / type, ascending / descending
- **Filter**: all / files only / directories only
- **+ button**: create new file or new folder

## Context Menus

- Right-click (desktop) or long-press (mobile) on items
- File actions: Edit, Rename, Duplicate, Delete
- Background actions: New File, New Folder

## Inline Rename

- Input field replaces filename text
- Enter or blur commits the rename
- Escape cancels

## File Operations

- **Create**: new file or folder via toolbar + button
- **Duplicate**: creates copy with `-copy` suffix
- **Delete**: with confirmation
- **Rename**: inline editing
- All operations update local state + send workspace protocol messages

## Deep Linking

- `[[file:...]]` links from chat (NB-20) navigate directly to the file in this tab

## File Editor

- Textarea-based editor
- Ctrl/Cmd+S to save
- Tab inserts spaces (not focus change)
- Dirty tracking (unsaved changes indicator)
- "Saved" feedback toast/badge

## Dependencies

- NB-18 (Navigation Overhaul) — tab infrastructure
