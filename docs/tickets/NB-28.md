# iOS — Enhanced Logs & Split Pane

> iOS counterpart of NB-24

**Scope:** Split layout with system monitor and enhanced log viewer

## Split Layout

- `GeometryReader` or `VSplitView` for top (monitor) / bottom (logs) split
- Pane cycling via header tap

## Log Viewer

- `.searchable` modifier for log filtering
- Color-coded log entries by type
- Copy and clear actions

## Dependencies

- NB-33 (iOS System Monitor) — embedded in top pane
- NB-28 (iOS Navigation) — Logs tab
